# Literature Search
## for CDKN2a

## Goal

COnduct a comprehensive literature search for this gene between 2021 and 2025.

## Implementation

- search PubMed and Europe PNC for CDKN2A and ARF
- use MCP servers for both resources
- create a Markdown file RESULTS.md with a paragraph per paper with which contains:
    - paper title
    - link to resource
    - a short description
    - DOI
- for each paper validated to ensure that this link points to this paper and so on
